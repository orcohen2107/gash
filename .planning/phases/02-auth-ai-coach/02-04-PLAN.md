**Design System (from DESIGN-SYSTEM.md):**
- User bubbles: surface-container-high (#20201f), aligned right (RTL)
- AI bubbles: primary-container (#00e3fd) with glassmorphism (60% opacity, 16px blur)
- Thinking indicator: AI bubble with bouncing dots
- Typography: body-lg (1rem, Inter/Heebo, 1.6 line-height), time label label-md (#adaaaa)

- Corner radius: 12px for bubbles

**Message Display Flow (from CONTEXT.md locked decision):**
1. User sends message → ChatInput captured, validated
2. Message sent to Edge Function (Plan 03)
3. While Claude processes: show TypingIndicator (bouncing dots) in AI bubble
4. Response arrives: remove thinking bubble, display response with TypewriterText animation
5. Typewriter speed: 40-50ms per character (locked decision)
6. After animation completes: message fully visible for long-press copy

**Copy-to-Clipboard (from CONTEXT.md):**
- Long-press any AI message bubble
- Show toast "העתק?" (Copy?)
- On confirmation: Clipboard.setString(content)
- Show toast "הועתק!" (Copied!)
- Use @react-native-clipboard/clipboard library

**RTL Rules (from CLAUDE.md):**
- textAlign: 'right' for all text inputs
- alignSelf: 'flex-end' for user bubbles, 'flex-start' for AI bubbles (RTL flips visually)
- No paddingLeft/Right, marginLeft/Right — use paddingStart/End, marginStart/End
- Test on physical device (Expo Go simulator may differ)
